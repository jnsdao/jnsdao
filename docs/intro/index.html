JNS奥德赛

从区块链小白到web3大师。

注: 什么是奥德赛？奥德赛是odyssey一词的音译。该词来源于荷马创作的两部重要的古希腊史诗之一《Odyssey》。因此odyssey一词常被用以指代一段漫长而冒险的旅程或经历。

联合举办方：JNS DAO, Jouleverse, Blockcoach.

🎁 任务：新手空投​

信息

任务名称：和组长取得联络，领取新手空投礼包（包含：JTI身份认证 + 0.005 J 燃料）。

  • 任务描述
  • 通关指南
  • 点亮技能
  • 通关秘笈
欢迎进入Jouleverse！你现在已经安全着陆在新手村。
现在，你需要首先领取到新手空投礼包。包含：JTI可信身份认证标识一枚（灵魂绑定NFT），以及 0.005 J 的免费燃料（gas）。
  1. 与12组组长中任意一位取得联络，请组长邀请你加入内测大群。
  2. 【选项A - 复杂】安装小狐狸钱包，保管好自己的助记词，配置好Jouleverse区块链网络，记下自己的Jouleverse区块链地址。（此步骤遇到困难请在分组群中交流）
  3. 【选项B - 简单】使用地址生成器生成一个冷钱包地址，在书本上抄写、保存好私钥，记下自己的Jouleverse区块链地址（稍后提供给组长）。（此步骤遇到困难请在分组群中交流）
  4. 向组长提供必要的信息，请组长协助提交审核后，发放新手空投礼包。
  5. 【A】查看小狐狸钱包，确认燃料到账。
  6. 【B】打开区块链浏览器，搜索自己的地址，打开自己的地址页面，查看地址剩余的燃料数量，以及JTI可信身份认证标识。
  7. 任务完成！
  1. ✅ 区块链钱包的安装和使用；
  2. ✅ 区块链钱包的网络设置；
  3. ✅ 区块链浏览器的使用。
  1. 如果在安装钱包方面遇到困难，可以先选择【选项B - 简单】路径，快速通关；让组长拉你进JNS拍卖群，直接参加JNS拍卖会。
  2. 据链友反馈，谷歌浏览器插件版小狐狸钱包、Firefox浏览器插件版小狐狸钱包、手机版小狐狸钱包App、手机版imtoken钱包App可以完成任务所需操作；
  3. 如何上网、如何安装，苹果手机海外id，等问题，请在分组小群里向其他链友请教。

🌍 任务：飞翔的J​

信息

任务名称：领取创世徽章「飞翔的J」NFT（灵魂绑定的）。

  • 任务描述
  • 通关指南
  • 点亮技能
  • 通关秘笈
现在，你已经领取到新手免费燃料（gas）了。接下来，你需要首先掌握必要的web3生存技能——合约交互，到链上领取创世徽章——一枚灵魂绑定NFT——「飞翔的J（Flying J）」。
  1. 打开小狐狸钱包（注：浏览器插件版）的高级设置，开启”十六进制数据”功能。
  2. 用小狐狸钱包向「飞翔的J」合约地址 0x045b997b5e05df9795985ab9e6720d94557255be 发起交易，金额 0 J，附加十六进制数据 0x4e71d92d 。
  3. 等待交易上链完成，用小狐狸钱包的“在浏览器中查看账户”功能，打开Jouleverse区块链浏览器查看自己钱包中已经领取到的「飞翔的J」徽章。
  4. 【可选】把Jouleverse区块链浏览器中展示的「飞翔的J」编号和徽章截图，发到内测群展示。
  5. 任务完成！
  1. ✅ 区块链合约交互；
  2. ✅ 区块链浏览器的使用。
  1. 这一关不是必须的！如果在安装钱包方面遇到困难，可以先跳过！让组长拉你进JNS拍卖群，直接参加JNS拍卖会。
  2. 据链友反馈，谷歌浏览器插件版小狐狸钱包、Firefox浏览器插件版小狐狸钱包、手机版imtoken钱包App可以完成任务所需操作；手机版小狐狸钱包App可能暂不支持十六进制数据功能。
  3. 遇到问题和困难，请在分组小群里向其他链友请教。

❤️ 任务：JNS拍卖会​

信息

任务名称：拍到自己心仪的JNS域名（至少1个），并绑定到自己的链地址。

  • 任务描述
  • 通关指南
  • 点亮技能
  • 常见问题
  • 成交排行榜
获得自己心仪的JNS藏品，成为JNS DAO的公民。
（注：JNS是J Name Service的缩写，是一种域名NFT。所有的JNS域名均以 .j 为后缀。）
  • 请你的组长把你拉进JNS拍卖群。
  • 在JNS拍卖群中预约出价。
  • 关注拍卖会开始时间的通知。
  • 准时参与JNS拍卖会，竞价胜出后，获得JNS域名。（拍卖参与方法）
  • 【可选】在Jouleverse区块链浏览器中查看收到的JNS拍品并截图，发到内测群展示。
  • 【可选】拍卖可以多次参与，获得多个中意的域名NFT藏品。
  • 【进阶】把你的JNS和你的地址绑定。（绑定方法）
  • 【可选】尝试和其他小伙伴交换你们的JNS藏品。
  • 任务完成。
  1. ✅ 社群互动；
  2. ✅ NFT的使用和交换。
  1. 问：JNS是永久有效的，还是（像互联网域名那样）需要定期续费的呢？
    答：JNS是链上NFT，所有权属于持有者，无需定期续费即可永久持有。
  2. 问：JNS的原始版权及授权协议是怎样的？
    答：JNS的原始版权属于JNS项目。授权协议初步考虑借鉴BAYC的极大化授权方式，把包括商业使用权在内的诸多权利都赋予JNS的持有者。
  3. 问：为什么每场拍卖只拍5个名字（不多拍一些呢）？
    答：实践表明，拍卖5个名字就需要耗时1-1.5小时，如果再多，时间更长，大家都会过于疲劳。
  4. 问：为什么不把预约列表中非热门名字直接免拍过，仅留下热门名字参拍呢？
    答：原因和上面类似，热门拍品需时较长，冷门则较短，搭配着来，时间比较合适；如果全是热门，时间会过长。
  5. 问：为什么限制每人每次只能提名预约一个名字，而不是放开无限量预约呢？
    答：为了防止预约列表过长，增加其他拍友阅读和识别、记忆上的困难。

❤️ 任务：JNSDAO投票治理​

信息

任务名称：参与JNS DAO的提案讨论和投票表决，获得POAP徽章（每个人的徽章颜色都不相同哦）。

  • 任务描述
  • 通关指南
  • 点亮技能
  • 常见问题
参与JNS DAO的提案讨论和治理投票，获得投票纪念徽章JNSVote POAP。
  • 积极参与到JNS DAO的日常讨论和任务中。参与或推动提案的讨论和表决。获得POAP徽章。
  • 任务完成。
  1. ✅ DAO的参与和贡献；
  2. ✅ 民主投票和社群自治启蒙。
  1. 问：JNS DAO的治理流程？
    答：三段式治理。前两步是链下的民意调查和共识投票，最后一步是链上投票。
  2. 问：什么是JNSVote POAP？
    答：POAP全称Proof-Of-Attendance Protocol，又称参与证明。JNSVote POAP是你参与JNS DAO链上治理投票的区块链证明。JNSVote POAP是灵魂绑定NFT。
  3. 问：JNS DAO链上投票表决的规则？
    答：链上投票通过提案的标准根据两个参数“赞成率”和“代表度”定义如下： 1. 赞成率超过 2/3 即约 66.67% 以上（投票者中，超过2/3多数赞成） 2. 赞成票的代表度超过 2/5 即约 40% 以上（赞成票投票者总共持有超过40%数量的JNS资产）
